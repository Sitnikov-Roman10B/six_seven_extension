Для запуска выполните следующие действия:
  1. Откройте Chrome(или любой другой браузер)
  2. Зайдите в раздел с расширениями
  3. Включите режим разработчика(в Chrome справа сверху)
  4. Нажмите "Загрузить распакованное расширение"(в Chrome слева сверху)
  5. Запустите появившееся расширение
Готово!
